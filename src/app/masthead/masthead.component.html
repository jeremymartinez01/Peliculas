Proyecto 6 - DAWM

Peli-espol es el proyecto 6 de la materia de Desarrollo de aplicaciones web y móviles, fue desarrollado por Jeremy Martinez, Viviana Falconí y Alejandra. Su propósito es brindar una variedad de peliculas icónicas usando el framework de Angular, peticiones a bases de datos y pulir las habilidades de css y html. Para este proyecto se siguió una serie de pasos basados en las guías del Ingeniero Allan Alvedaño
